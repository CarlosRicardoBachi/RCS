# 0.3.10.2.32 — HOTFIX_ToolModeToolbar

## Objetivo
Restaurar arranque de la app tras 0.3.10.2.30/31: la toolbar intentaba usar `ToolMode.MOVE`, que no existe en `rcs/core/tool_mode.py` (los modos válidos son SELECT/PICK/ZOOM/ROTATE/SCALE/PAN).

## Cambios
- Toolbar “Herramientas”:
  - “Seleccionar” ahora usa `ToolMode.PICK` (solo seleccionar; no arrastra).
  - “Mover” ahora usa `ToolMode.SELECT` (seleccionar + mover con drag).
- Persistencia / estado inicial:
  - el modo inicial ahora se restaura comparando `action.data()==tool_mode` (no el texto del botón).

## DoD
1. `python -m rcs.app` inicia sin excepciones.
2. Cambiar entre herramientas (Seleccionar/Mover/Zoom/Pan/Rotar/Escalar) actualiza el statusbar y responde.
3. Cerrar y reabrir: conserva el modo elegido (settings).
