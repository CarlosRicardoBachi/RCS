# Patch 0.3.10.2.62 — TOOLBOX UX (Space=Pan temporal + cursores dedicados)

## Objetivo
Hacer que la toolbox se sienta más “editor pro”: pan rápido sin romper tu flujo y señales visuales claras de qué herramienta está activa.

## Cambios
### 1) Space = PAN temporal (hold)
- **Mantener presionado Space** activa el modo **PAN** (ScrollHandDrag).
- **Soltar Space** restaura el tool anterior.
- Guardas:
  - No se activa si hay drag activo de **Rotate** o **Scale** (evita interferencias durante transformaciones).
  - Se ignoran repeats (`event.isAutoRepeat()`), para no “spamear” el switch.

### 2) Cursores dedicados por tool
- **PAN**: mano (OpenHandCursor).
- **ROTATE**: cursor custom con flecha curva (QCursor generado en runtime con QPainter).
  - Fallback: CrossCursor si el cursor custom no puede construirse.
- **SCALE**: cursor de tamaño (SizeAllCursor).

## Pruebas manuales rápidas
1. Abrir un proyecto y dejar tool en **Seleccionar**.
2. Mantener **Space** y arrastrar: el canvas debe panear.
3. Soltar **Space**: vuelve a Seleccionar.
4. Cambiar a **Rotar** y **Escalar**: verificar que el cursor cambie y que no quede “pegado” al alternar.
