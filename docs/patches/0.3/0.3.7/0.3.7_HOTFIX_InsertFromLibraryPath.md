# 0.3.7 — HOTFIX (Inserción desde biblioteca / paths)

Fecha: 2026-01-16

## Síntoma
- Doble click en un thumbnail → el objeto no aparece en el lienzo.
- En algunos casos “aparecía” al cambiar el tema del lienzo (forzaba redraw), lo que era una pista falsa.

## Causa probable
Resolución inconsistente de rutas:
- `source` relativo sin anclar correctamente a `components_root`.
- O bien mezcla de paths Windows (`\`) y paths relativos dentro del proyecto.

## Fix (intención)
- Normalizar `source` como relativo a la biblioteca.
- Al instanciar el item/render, resolver a path absoluto confiable.

## Riesgo / WATCH
- Esta zona es propensa a regresión cuando se toca cámara (zoom/pan/center) o settings.
- Ver `docs/OBSTACLES.md` (OBS-004).
