# 0.3.10.2.61 — HOTFIX CanvasView Restore

## Objetivo
Restaurar el comportamiento estable del canvas (lienzo/hoja/grilla) y la inserción desde biblioteca, evitando estados “a medias” por parches que dejaron **CanvasView** sin API completa.

## Síntomas que corrige
- Error: `AttributeError: 'CanvasView' object has no attribute 'set_project'`
- Error: `AttributeError: 'CanvasView' object has no attribute 'insert_svg_from_library'`
- Lienzo/hoja no visible o canvas “vacío” pese a iniciar la UI.

- `rcs/ui/canvas_view.py`: **reemplazado** por snapshot estable de 0.3.10.2.56 (mantiene rotate gizmo, handles de escala, overlay amarillo, zoom suavizado).
