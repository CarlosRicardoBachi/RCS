# 0.3.10.2.67 — HOTFIX MainWindow ProjectSet Compat

## Objetivo
Desbloquear **Importar GMPR** cuando la UI llama `self.set_project(...)` pero `MainWindow` no expone ese método (crash).
Además, eliminar otro crash por llamados a `_refresh_title()` inexistente en esta rama.

## Cambios (mínimos)
- `MainWindow.set_project(project)`:
  - asigna `self._project = project`
  - llama `self._refresh_project()` (rebuild canvas + panels + título)
- `MainWindow._refresh_title()`:
  - alias a `self._update_title()` (compat)

## Criterios de aceptación
1) `Archivo → Importar GMPR…` ya no falla con:
   - `AttributeError: 'MainWindow' object has no attribute 'set_project'`
2) La app no falla por `_refresh_title()` al importar SVG externo o GMPR.
3) No se cambió lógica de GMPR I/O: solo se agregó compat en UI.

## Riesgos / Notas
- Cambio encapsulado en `MainWindow` (2 métodos pequeños).
- No modifica CanvasView ni core de transforms.
