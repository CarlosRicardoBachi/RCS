# 0.3.10.2.59 — HOTFIX: Regresión CanvasView API

## Contexto
En algunos entornos quedó un `rcs/ui/canvas_view.py` desfasado o parcialmente aplicado, provocando crash al iniciar:
- `AttributeError: 'CanvasView' object has no attribute '_rerender_svg_previews'`
- `AttributeError: 'CanvasView' object has no attribute 'set_project'`

## Cambios
- **Restaura/asegura API esperada por `MainWindow`**:
  - `CanvasView.set_project(...)`
  - `CanvasView._rerender_svg_previews()`
- **Guard en `set_preview_style()`**: si el rerender no existe, se registra **un warning de 1 línea** y la app continúa.

## Criterio de aceptación
1. `python -m rcs.app` inicia sin crash.
2. Cambiar “preview stroke/outline” desde UI **no crashea**.
3. `set_project()` se ejecuta al iniciar y la UI queda operativa (insertar SVG, seleccionar, etc.).
