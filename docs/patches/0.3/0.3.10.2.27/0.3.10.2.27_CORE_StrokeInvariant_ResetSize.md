# 0.3.10.2.27 — CORE — Stroke fijo en preview SVG + Restaurar tamaño

## Objetivo
1. Evitar que el grosor de líneas (stroke + halo/contorno) de los SVG *en el preview raster* se “engorde” cuando el usuario escala el objeto.
2. Agregar una acción/botón para volver el objeto a su tamaño original (escala 1:1).

## Cambios principales
- Preview SVG (raster):
  - `_render_svg_preview_pixmap(...)` ahora recibe `obj_scale` y compensa el grosor dividiendo el `stroke_px`/`outline_px` por la escala actual del objeto.
  - Al escalar (modo SCALE con rueda), se re-renderiza **solo** el preview del objeto afectado para mantener el grosor visual constante.
  - Re-render global (`zoom/theme`) delega a un re-render por objeto y respeta `svg_fit_content` (si estaba activado, vuelve a recortar al contenido).

- UI: “Restaurar tamaño (1:1)”
  - Canvas (click derecho): opción **Restaurar tamaño (1:1)**.
  - Panel de Objetos: botón **1:1**.
  - Menú **Objeto**: acción **Restaurar tamaño (1:1)** con atajo `Ctrl+1`.

## Pruebas manuales rápidas
1. Insertar un SVG con líneas finas.
2. Poner herramienta **SCALE** y escalar con la rueda:
   - El tamaño del objeto cambia.
   - El grosor aparente del stroke **no** debería engordar.
3. Click derecho → **Restaurar tamaño (1:1)** (o botón **1:1** en el panel):
   - El objeto vuelve a escala 1.0.
4. Activar **Ajustar al contenido** y luego cambiar zoom:
   - El recorte debería mantenerse.
