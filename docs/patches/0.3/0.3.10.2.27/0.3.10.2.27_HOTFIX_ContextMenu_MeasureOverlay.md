# 0.3.10.2.27 — HOTFIX — Context Menu + Measure Overlay

Este hotfix corrige dos fallos que bloqueaban UX:

## 1) Overlay de medida amarilla no aparece al escalar
- Causa: `NameError` por `QFontMetrics` no importado.
- Fix: importar `QFontMetrics` desde `PySide6.QtGui`.
- Extra: proteger la rutina de overlay en `drawForeground()` con `try/finally` para no dejar `painter.save()` sin `restore()`.

## 2) Menú contextual (botón derecho) no aparece
- Causa: `AttributeError` en `contextMenuEvent()` por falta del método `_paste_size_to_selection` referenciado por una `QAction`.
- Fix: implementar `_paste_size_to_selection()` como wrapper que toma `self._clipboard_size_mm=(w_mm,h_mm)` y llama a `apply_size_to_selection_mm()`.

## Verificación
- Abrir app → seleccionar objeto → botón derecho → menú aparece.
- Escalar objeto → overlay de medidas aparece sin crashear.
