# 0.3.10.2.29 — CORE — Preview SVG: escala de grosor ajustada (rango más fino)

## Objetivo
Ajustar el rango de **Grosor de línea / Contorno** del preview (menú **Ver → Vista previa**) para que los niveles bajos no arranquen “gruesos”.

En la práctica: el nivel **1** debía sentirse realmente fino; y los demás niveles escalar desde ahí sin que el 1..6 parezca un “mazazo” desde el arranque.

## Cambio
- Se introduce una **escala global** aplicada a los valores del menú de preview (línea + contorno).
- Default: **x0.5** sobre el nivel seleccionado.
  - Ej: nivel 1 ⇒ ~0.5px lógico (antes ~1px)
  - nivel 6 ⇒ ~3px lógico (antes ~6px)

## Tuning (opcional)
- `RCS_CANVAS_PREVIEW_THICK_SCALE` (default: `0.5`, clamp `0.1..4.0`)
  - Permite afinar el rango sin tocar el código.

## Pruebas manuales sugeridas
1. Abrir proyecto con SVG de contorno (por ej. llavero).
2. Escalar un objeto 5×–10×.
3. Ir a **Ver → Vista previa → Grosor de línea** y probar **1..6**.
4. Verificar:
   - El nivel 1 se ve claramente más fino que antes.
   - El crecimiento 1→6 mantiene la progresión sin saltos raros.
   - No afecta exportación ni datos del SVG (solo preview).
