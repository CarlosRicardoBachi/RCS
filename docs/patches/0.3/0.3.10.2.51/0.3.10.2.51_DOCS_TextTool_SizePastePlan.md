Fecha: 2026-01-26

> Este patch es **documental**: consolida el plan y deja el proyecto alineado para retomar en una nueva sesión (IA o humano) sin “explicaciones” previas.

## Qué se agrega/actualiza
### 1) Text Tool (placeholder → SmartText)
- Se define la arquitectura y etapas del módulo de texto en `docs/RCS_TEXT_TOOL.md`.
- Se formalizan 2 modos clave para llaveros:
  - **Modo `continuous` (letra continuada)**: puenteado + “Islas como grabado” (puntos/acentos no quedan sueltos).
  - **Modo `plate` (contorno + letras dentro)**: genera `PLATE` abrazando + `LETTERS`, con parámetro `outer_margin_mm`.
- Decisiones cerradas:
  - `seat_clearance_mm`: no se usa.
  - `shadow_offset_mm`: se omite (opcional a futuro, no prioritario).
