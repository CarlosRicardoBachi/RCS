# 0.3.10.2.51 — HOTFIX: Size Paste blindado + Text Tool placeholder avanzado + HUD ángulo rotación

## Objetivo A — Copiar/Pegar tamaño (Canvas)
### Cambios
- **Validación dura del clipboard interno**:
  - Rechaza `None`, `bool`, no-numérico, `NaN/Inf`, `<= 0`.
  - Si llega un valor inválido, se **borra** y se deja un **log de 1 línea** (para rastrear origen sin spamear).
- **Rotados**: la medida se toma desde `sceneBoundingRect()` (bbox visible real), así que copiar/pegar “respeta lo que ves”.
- **Multi-selección**: pegar tamaño ahora aplica **escalado real por grupo**:
  - bbox unido + pivot en el centro del grupo
  - escala tamaño **y** posiciones relativas (se “estira/encoge” el conjunto como una sola pieza)
- **Pegar solo ancho/alto** ahora mantiene relación de aspecto (escala uniforme).

### Archivos tocados
- `rcs/ui/canvas_view.py`

## Objetivo B — Text Tool placeholder (UI)
### Cambios
- Preview ahora soporta **multilínea real** (usando `\n`).
- Se agregó **interlineado** (`line_spacing`, multiplicador).
- Se agregaron **modos** (metadatos por ahora):
  - `normal`
  - `placa` (incluye `outer_margin_mm` y maqueta simple en preview)
  - `continuado` (incluye `engrave_islands` default ON)

### Archivos tocados
- `rcs/ui/text_tool_panel.py`
- `rcs/ui/main_window.py` (acción de toolbar para mostrar/ocultar dock de texto)

## Extra — HUD de ángulo al rotar
- Al rotar con la rueda, se muestra un overlay con el **ángulo actual** (misma estética que el overlay de medidas).
- `Shift` aplica rotación **fina** (0.5° por notch).
