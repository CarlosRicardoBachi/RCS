# Patch Note — 0.3.10.2.56 — UI_ZoomSmoother_Wheel

## Objetivo
El zoom por rueda estaba “agresivo” (saltos grandes por notch). Este hotfix suaviza la curva de zoom sin cambiar la lógica del resto de herramientas.

## Cambios
- `CanvasView.wheelEvent()`:
  - Zoom por rueda (ToolMode.ZOOM) ahora aplica un multiplicador de suavizado:
    - **Default**: ~40% menos agresivo.
    - **Shift**: ~60% menos agresivo (ajuste fino).
  - `Ctrl + rueda` (modo Select) respeta los mismos factores para consistencia.

## Criterio de aceptación
- Con ToolMode.ZOOM: un notch de rueda produce un cambio perceptiblemente menor que antes.
- Manteniendo Shift mientras se hace zoom: el cambio es aún más pequeño que en default.
- Con Ctrl+rueda en modo Select: el comportamiento coincide con ToolMode.ZOOM.
- No afecta la rueda cuando ToolMode es ROTATE/SCALE (ahí la rueda sigue actuando sobre el objeto).
