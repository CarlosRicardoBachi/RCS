# 0.3.10 — Bloque 3I: Encuadre de vista (selección/todo/hoja)

**Fecha:** 2026-01-17  
**Tipo:** feature  
**Estado:** stable

## Objetivo
Agregar acciones de **encuadre** para recuperar el foco rápidamente:
- **Encuadrar selección**
- **Encuadrar todo** (solo objetos, no la hoja)
- **Ver hoja** (reset de vista al canvas)

Esto cierra el pendiente recurrente de “me perdí en el lienzo” sin tocar el pipeline de SVG.

## Cambios
### UI
- Menú **Ver**:
  - **Encuadrar selección** → `F`
  - **Encuadrar todo** → `Shift+F`
  - **Ver hoja** → `Ctrl+Shift+0`

### Canvas
- Nuevo encuadre basado en bounding boxes reales:
  - usa `sceneBoundingRect()` de items objeto
  - aplica `fitInView(..., KeepAspectRatio)`
  - sincroniza zoom interno con `_sync_zoom_from_view_transform()`
