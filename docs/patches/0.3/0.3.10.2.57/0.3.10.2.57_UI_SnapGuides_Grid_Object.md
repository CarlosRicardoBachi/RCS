# 0.3.10.2.57 — UI_SnapGuides_Grid_Object

## Objetivo
Agregar snap utilizable para flujo “láser/CNC”: grilla en mm + guías visuales (amarillas) cuando la selección engancha con otras geometrías.

## Alcance
- Solo **ToolMode.SELECT** (mover/drag).
- No altera formatos de proyecto ni pipeline de importación/exportación.
- Sin cambios fuera del patch.

## UX
- **S**: toggle Snap ON/OFF (mensaje overlay).
- **Shift + drag**: step fino (0.1 mm).
- **Alt + drag**: snap temporal OFF (por ese drag).
- Guías amarillas: líneas vertical/horizontal al enganchar por **borde** o **centro** (X/Y).

## Implementación (resumen)
- Snap post-move: después de `super().mouseMoveEvent`, cuantiza la bbox unida de la selección.
- Grid snap: anclado al origen del canvas (mm).
- Object snap: considera líneas candidatas (left/center/right, top/center/bottom) de objetos **no seleccionados** y del canvas.
- Prioridad: object-snap > grid-snap.
- Threshold en pantalla: ~12px (convertido a mm por zoom).

## Criterio de aceptación
1) Con Snap ON, al mover un objeto, se alinea a grilla 1.0 mm.
2) Con Shift durante drag, la grilla pasa a 0.1 mm (más fino).
3) Con Alt durante drag, no hay snap (se mueve libre).
4) Al acercarse a borde/centro de otro objeto, aparece guía amarilla y el objeto engancha (si está dentro del threshold).
5) No rompe: scale/rotate gizmos, overlays, import SVG, copy/paste size.
