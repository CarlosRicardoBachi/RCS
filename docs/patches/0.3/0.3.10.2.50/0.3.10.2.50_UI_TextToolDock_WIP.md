# 0.3.10.2.50 — UI Dock Texto (WIP): maquetado + preview + metadatos

## Objetivo
Arrancar la **maqueta** de la nueva herramienta de texto (la que dibujaste en Paint) sin comprometer todavía el pipeline de objetos/vector/export.

Este patch no crea "texto como objeto" todavía; solo deja el **esqueleto UI** listo para iterar rápido.

## Qué incluye
- Nuevo dock **Texto** (Ver → Paneles → Texto).
- Controles base:
  - Texto (multilínea)
  - Fuente (combo) + Bold/Italic
  - Alineación Izq/Centro/Der
  - Dimensiones (mm)
  - Estiramiento horizontal/vertical (placeholder)
- Preview rápido del primer renglón (render a pixmap) para validar look & feel.
- Salida JSON de metadatos (placeholder) para ir estabilizando el schema del futuro SmartText.
- Botón **Insertar (WIP)**: hoy solo emite metadatos y actualiza status (sin tocar el lienzo).

## Notas de diseño
- El dock **no** crea QGraphicsItems ni paths en el canvas por ahora, para evitar romper selección/export.
- El preview está hecho a propósito “simple y barato”: es para maquetado.
