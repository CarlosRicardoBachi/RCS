# Patch — RCS 0.3.10.2.55 (UI Rotate Gizmo + Snap 5°)

## Qué cambia
- ToolMode.ROTATE agrega un **gizmo** (handle circular) arriba del bbox de la selección.
- Click + drag sobre ese handle rota **toda la selección** alrededor del centro del bbox.
- Rotación con *snap* por defecto a **5°** (más “CNC-friendly”).
- Con **Shift** durante el drag: *snap* fino a **1°**.
- ToolMode.ZOOM intenta usar cursor tipo **lupita** (ZoomInCursor) cuando Qt lo soporta.

## Notas de implementación (para no romper cosas)
- La rotación de grupo se hace en dos pasos por objeto:
  1) aplicar `rotation_deg` al modelo (y sincronizar item),
  2) recolocar el **centro** del item rotado alrededor del pivot para evitar drift y mantener el grupo coherente.

## Cómo probar
1. Seleccionar 1 objeto → ROTATE → aparece el handle “⟲”.
2. Click+drag sobre el handle:
   - Overlay muestra ΔÁngulo en amarillo.
   - El objeto rota con saltos de 5°.
3. Mantener Shift y repetir: saltos de 1°.
4. Multi-selección (2+ objetos):
   - click+drag rota como grupo (se conserva la geometría relativa).
5. ZOOM:
   - activar herramienta y verificar cursor de lupa si está disponible en tu Qt.
