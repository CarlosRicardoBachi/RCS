# 0.3.10.2.71 — HOTFIX GMPR Import: Fix NameError `canvas_w_mm`

## Problema
Al importar GMPR, el flujo de reconstrucción de rasters llamaba a `_gmpr_raster_transform_to_rcs_transform(...)` pasando `canvas_mm=(canvas_w_mm, canvas_h_mm)` sin definir esas variables.

Resultado: el import falla con **NameError** y no llega a restaurar rasters.

## Fix
En `gmpr_to_project()` se define siempre:

- `canvas_w_mm, canvas_h_mm = prj.canvas_mm`

`Project.canvas_mm` siempre existe (tiene default), incluso si no se pudo inferir canvas desde el SVG embebido.
