# 0.3.10.2.17 — HOTFIX — Restore version constants (Windows-safe)

## Problema
Después de aplicar 0.3.10.2.16, RCS no iniciaba y arrojaba:

- `ImportError: cannot import name 'SCHEMA_VERSION' from 'rcs.core.version'`

## Causa
`rcs/core/version.py` fue sobrescrito con una versión recortada que solo definía `APP_NAME` y `APP_VERSION`, eliminando:
- `SCHEMA_VERSION`
- `DEFAULT_CANVAS_MM`
- `DEFAULT_GRID_MM`
- `APP_SHORT`

## Fix
Se restaura el contenido completo de `rcs/core/version.py` con las constantes requeridas, manteniendo el resto del parche 0.3.10.2.16 intacto.

## Archivos
- `rcs/core/version.py`

## Verificación
En `C:\PROYECTOS\RCS` (con venv activa):

```bat
python -c "from rcs.core.version import APP_VERSION, SCHEMA_VERSION; print(APP_VERSION, SCHEMA_VERSION)"
python -m rcs.app
```

Esperado:
- Imprime `0.3.10.2.17 1`
- La app inicia sin ImportError
