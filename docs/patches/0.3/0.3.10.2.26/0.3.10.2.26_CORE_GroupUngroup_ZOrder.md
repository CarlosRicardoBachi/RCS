# 0.3.10.2.26 — CORE — Group/Ungroup + Z-order por grupo

## Objetivo
Agregar agrupación lógica de objetos para que:
- Seleccionar 1 miembro seleccione automáticamente el grupo completo.
- Z-order (frente/fondo + subir/bajar un nivel) trate al grupo como un solo bloque.
- Agrupar/Desagrupar esté disponible en el panel de objetos (derecha), en el canvas (click derecho) y en el menú **Objeto**.

## Cambios principales
- Nuevo campo opcional: `SceneObject.group_id` (persistente en proyecto).
- Canvas:
  - Auto-expansión de selección por `group_id`.
  - `Agrupar` / `Desagrupar`.
  - Normalización local de Z al agrupar para crear un bloque contiguo (sin tocar objetos fuera del rango entre min/max del grupo).
  - Clipboard: al pegar, se remapea `group_id` (para que el grupo pegado sea nuevo y no quede enlazado con el original).
- UI:
  - Panel de Objetos: botones **G** (agrupar) y **U** (desagrupar) + opciones en menú contextual.
  - Menú **Objeto**: atajos `Ctrl+G` (agrupar) y `Ctrl+Shift+G` (desagrupar).

## Pruebas manuales rápidas
1. Crear 3 objetos (rect, text, svg).
2. Seleccionar 2 → **G** (o click derecho → Agrupar).
3. Click en 1 miembro: deben seleccionarse los 2.
4. Z-order:
   - Subir/Bajar un nivel: el grupo debe moverse como bloque.
   - Traer/Enviar: idem.
5. Copiar/Pegar el grupo: el grupo pegado debe seguir agrupado, pero no debe quedar “atado” al original (desagrupar uno no afecta al otro).
