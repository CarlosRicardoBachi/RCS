# 0.3.10.2.73 — HOTFIX GMPR Base SVG Visible

## Problema
- El import GMPR podía romper el arranque por un `SyntaxError` en `rcs/ui/canvas_view.py`.
- El SVG base embebido del GMPR podía quedar invisible en tema oscuro (stroke negro + opacidad baja).

## Fix
- Se reescribe el render del SVG base GMPR para evitar el `SyntaxError`.
- Se inyecta CSS *en memoria* para forzar alto contraste (fill none + stroke claro) en el preview de base.
- Opacidad configurable por env.

## Env (opcionales)
- `RCS_GMPR_BASE_OPACITY` (0.05..1.0) — default `0.55`
- `RCS_GMPR_BASE_STROKE_PX` (0.1..10.0) — default `1.25`
- `RCS_GMPR_BASE_STROKE_USER` — fuerza stroke en unidades del SVG (override)
- `RCS_GMPR_BASE_STROKE_COLOR` — default `#ffffff`

## Test
1) `python -m py_compile rcs/ui/canvas_view.py`
2) `python -m rcs.app`
3) Importar `.GMPR` y verificar que el SVG base se ve como referencia.
