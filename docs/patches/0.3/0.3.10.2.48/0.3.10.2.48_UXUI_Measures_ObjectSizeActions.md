# 0.3.10.2.48 — UX/UI — Medidas (mm) + acciones de tamaño

## Problema
En la cadena de hotfixes posteriores a 0.3.10.2.44 se perdió el HUD de medidas (mm) y algunas acciones UI.

## Cambios
### CanvasView
- HUD fijo (amarillo) con **tamaño del lienzo** en mm (esquina superior izquierda).
- HUD temporal (amarillo) con **tamaño del objeto/selección** (W x H en mm):
  - se muestra al **escalar** un objeto
  - se puede invocar con la tecla **V** (3s)
- Menú contextual en objetos:
  - **Copiar**: alto / ancho / ancho+alto
  - **Pegar**: tamaño (ancho+alto)

### MainWindow
- Toolbar: botón **Medida objeto** → diálogo para ajustar **ancho/alto (mm)**
  - incluye checkbox para **mantener relación de aspecto**

## Test rápido
1. Abrir RCS y verificar HUD de lienzo (W x H mm).
2. Seleccionar objeto, escalar: aparece HUD temporal con W x H.
3. Presionar **V** con un objeto seleccionado: HUD por 3 segundos.
4. Click derecho objeto → Copiar ancho+alto → en otro objeto Pegar tamaño.
5. Toolbar → **Medida objeto**: setear ancho/alto y confirmar.
