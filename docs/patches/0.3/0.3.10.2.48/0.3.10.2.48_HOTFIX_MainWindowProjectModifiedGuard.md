# 0.3.10.2.48 — HOTFIX MainWindow ProjectModified Guard

## Problema
En algunos merges/overrides de parches, el método `MainWindow._on_project_modified` puede desaparecer o quedar fuera de la clase, provocando crash al conectar la señal:

- `self._canvas.project_modified.connect(self._on_project_modified)`

## Fix
- Se agrega guard defensivo en `MainWindow._build_ui()`:
  - Si existe `_on_project_modified`, conecta normal.
  - Si no existe, conecta a un fallback que solo refresca el título.

## DoD
- `python -m rcs.app` levanta sin crash.
- Modificar el proyecto dispara actualización del título (dirty) aunque falte el handler.
