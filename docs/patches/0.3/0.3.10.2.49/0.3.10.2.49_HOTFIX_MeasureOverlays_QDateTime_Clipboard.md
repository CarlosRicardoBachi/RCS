# 0.3.10.2.49 — HOTFIX MeasureOverlays: QDateTime + Clipboard Size

## Problema
Al activar overlays de medidas:
- `NameError: QDateTime is not defined` en `CanvasView.drawForeground()` rompía el pintado.
- En menú contextual: `_copied_size_mm` podía ser `None`, causando `AttributeError: 'NoneType' object has no attribute 'get'` y el menú “parpadeaba” / no aparecía.

## Fix
- Importa `QDateTime` desde `PySide6.QtCore`.
- Inicializa `self._copied_size_mm` como `{}` (nunca `None`) y agrega guard seguro en `setEnabled()` del action “Pegar tamaño del objeto”.
