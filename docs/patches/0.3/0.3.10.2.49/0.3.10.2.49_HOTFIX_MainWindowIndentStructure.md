# 0.3.10.2.49 — HOTFIX MainWindowIndentStructure

## Problema
El archivo `rcs/ui/main_window.py` tenía un bloque con indentación incorrecta:
- `_action_image_size_mm` estaba definido a nivel módulo (sin indentación de clase).
- Las acciones siguientes (`_action_canvas_auto`, etc.) quedaron anidadas dentro de esa función.
Resultado: `MainWindow` no tenía los handlers esperados y la app crasheaba al iniciar.

## Fix
- Reindenta el bloque de `_action_image_size_mm` para que sea método real de `MainWindow`.
- Deja las acciones siguientes como métodos hermanos (no anidados).

## Verificación
```bat
python -c "from rcs.core.version import APP_VERSION; print(APP_VERSION)"
python -m rcs.app
```
Debe iniciar sin `AttributeError`/`SyntaxError` en `main_window.py`.
