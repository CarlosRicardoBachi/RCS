# RCS 0.3.10.2.52 — HOTFIX — HUD overlays + Scale handles

## Qué se arregla
- Se corrige el error que rompía el render de overlays (AttributeError por `_overlay_aux_*` no inicializados), lo que hacía “desaparecer” los textos amarillos y spamear tracebacks en consola.
- El HUD vuelve a mostrar información clave:
  - **Hoja** (tamaño del área/plancha en escena)
  - **Lienzo** (canvas del proyecto)
  - **Zoom**
  - **Objeto** (bbox de selección) durante escalado
  - **Ángulo** durante rotación

## Qué se agrega
- **ToolMode.SCALE**: aparecen **4 handles (puntitos)** en las esquinas del bbox de selección.
  - Click + drag en una esquina → escala desde la esquina opuesta (pivot fijo).
  - **Shift** durante drag → mantiene aspecto (escala por diagonal).

## Archivos tocados
- `rcs/ui/canvas_view.py`

## Pruebas manuales
1. Abrir RCS. Verificar que NO hay tracebacks repetidos en consola.
2. Seleccionar un objeto y entrar a **Escalar**:
   - Se ven los 4 puntitos en las esquinas.
   - Drag de una esquina cambia tamaño.
   - Shift+drag mantiene proporción.
3. Entrar a **Rotar** y usar rueda:
   - El HUD muestra grados (ángulo) durante unos segundos.
4. Usar rueda en **Escalar**:
   - El HUD muestra tamaño del objeto seleccionado.
