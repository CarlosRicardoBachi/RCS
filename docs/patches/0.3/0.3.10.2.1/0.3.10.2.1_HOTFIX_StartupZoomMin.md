# 0.3.10.2.1 (HOTFIX) — Arranque: fix crash en zoom mínimo adaptativo

## Arreglo
En el cálculo de límites de zoom, `_sheet_rect` puede ser un `QRectF` o (en algunos flujos) una referencia a función/método.
Este parche normaliza ese valor: si es **callable**, se invoca; si no se puede invocar, se trata como no disponible.

## Archivos
- `rcs/ui/canvas_view.py`

## Pruebas
1. Ejecutar `python -m rcs.app`.
2. Redimensionar la ventana varias veces.
3. Usar el slider de zoom y la rueda: no debe permitir zoom-out extremo si la hoja ya entra.

## Notas
- No cambia el schema `.RCS`.
- Parche seguro: solo afecta el cálculo del mínimo de zoom.
