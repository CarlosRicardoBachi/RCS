# 0.3.10.2.31 — HOTFIX_ToolbarStyleCrash

**Fecha:** 2026-01-23  
**Tipo:** HOTFIX (startup crash)

## Problema
Al iniciar, la app crasheaba en `_build_toolbar()` con:
`AttributeError: 'ToolButtonStyle' object has no attribute 'iconSize'`.

Causa: `_toolbar_qt_style()` devuelve un `Qt.ToolButtonStyle` (enum), pero el código lo trataba como un objeto con
atributos `iconSize` y `toolButtonStyle`.

## Solución
- Separación correcta:
  - `tb.setToolButtonStyle(<Qt.ToolButtonStyle>)`
  - `tb.setIconSize(<QSize>)`
- Se agrega helper `_toolbar_icon_size()` con override por env:
  - `RCS_TOOLBAR_ICON_SIZE` = `20` o `20x20` (default 22).
