# 0.3.10.2.19 (HOTFIX) — Import de QByteArray en renderer Qt

## Problema
Al insertar ciertos SVG, el renderer Qt podía intentar construir `QSvgRenderer(QByteArray(svg_bytes))` pero faltaba el import de `QByteArray`, disparando `NameError`.

## Cambio
Se agrega el import faltante para que el path de render por bytes funcione correctamente.

## Archivos tocados
- `rcs/ui/canvas_view.py`

## Prueba rápida
1. Abrí RCS.
2. Insertá un SVG desde la biblioteca.
3. Verificá que ya no aparece el error `name 'QByteArray' is not defined`.
