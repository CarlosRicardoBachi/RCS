# 0.3.10.2.46 — HOTFIX MainWindow Indent

## Problema
La app no iniciaba por `IndentationError` en `rcs/ui/main_window.py` (bloque de acción "Ajustar tamaño de imagen" quedó sin indentación dentro del builder de toolbars).

## Fix
- Reindentado del bloque `act_img_size` para que quede dentro de `_build_toolbars()`.
- No cambia comportamiento, solo corrige el crash de arranque.
