RusticCreadorSvg (RCS) — Bloque 0 (v0.0.1)
=============================================

Requisitos
- Python 3.11+
- PySide6
- pytest (solo para tests)

Ejecución
- `python -m rcs.app`
  o
- `python rcs/app.py`

Bloque 0: esqueleto estable
- Ventana principal + menú Archivo
- Nuevo / Abrir / Guardar / Guardar como / Salir
- Formato .RCS (JSON legible)
- Logging a consola + logs/rcs.log
- Tests mínimos de save/load

Nota:
- El lienzo (QGraphicsView) y SVG importer/exporter están como placeholders permitidos SOLO en v0.0.x.
