# RCS 0.3.3 - Checklist manual

> Nota: esta checklist se usa para cada entrega 0.3.x. Si falla algo ya probado: HOTFIX antes de seguir.

## 1) Toolbar (modo exclusivo)
- [ ] Abrir RCS.
- [ ] Cambiar herramienta a **Zoom**: la rueda hace zoom.
- [ ] Cambiar herramienta a **Rotar**: la rueda rota el objeto seleccionado.
- [ ] Cambiar herramienta a **Escalar**: la rueda escala el objeto seleccionado.
- [ ] Volver a **Seleccionar**: la rueda hace scroll y **Ctrl+rueda** zoom.

## 2) Zoom slider
- [ ] Mover slider: cambia el zoom.
- [ ] Hacer zoom con rueda (modo Zoom): slider refleja el cambio.

## 3) Canvas size (mm)
- [ ] Menú **Lienzo > Tamaño en mm...**
- [ ] Cambiar tamaño y aceptar: el rectángulo del lienzo cambia.
- [ ] Guardar y reabrir: el tamaño se conserva.

## 4) Editar objetos
- [ ] Insertar SVG.
- [ ] Seleccionar + mover: guardar y reabrir => mantiene posición.
- [ ] Modo Rotar: rueda rota; guardar y reabrir => mantiene rotación.
- [ ] Modo Escalar: rueda escala; guardar y reabrir => mantiene escala.
- [ ] Del elimina selección.
- [ ] Ctrl+C / Ctrl+V duplica el objeto.
- [ ] Flechas: mueven selección (1 mm). Shift+Flecha (10 mm). Alt+Flecha (0.1 mm).
- [ ] Ctrl+D: duplica selección (equivalente a copiar/pegar).

## 5) Tool "Seleccionar (solo)"
- [ ] Cambiar a **Seleccionar**: click selecciona, arrastrar NO mueve.
- [ ] Cambiar a **Mover**: arrastrar mueve.

## 6) Toolbar acciones rápidas
- [ ] Botón **Duplicar**: duplica selección.
- [ ] Botón **Lienzo**: abre el diálogo de tamaño en mm.
